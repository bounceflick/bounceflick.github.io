Now this is a funky enumerable! Cycle is a enumberable who can create an infinite loop if not careful, but who's main use is to loop data sets. Useful examples of this include recurring data like dates, menus, etc. Below is a good example.

     days_of_week = %w{Monday Tuesday Wednesday Thursday Friday Saturday Sunday}
     => ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"]

     # days of the week in a repeating sequence.
     days = days_of_week.cycle
     => #

     # Enumerator's "next" method gives us the next day of the week
     days.next
     => "Monday"

     days.next
     => "Tuesday"

     days.next
     => "Wednesday"

     # Enumerator's "rewind" method resets the enumerator back
     # to the first item
     days.rewind
     => # "Monday"

     ...

     # If you keep going, the enumerator will "wrap around" back
     # to the beginning
     days.next
     => "Saturday"

     days.next
     => "Sunday"

     days.next
     => "Monday"

What this is great for is recurring items, where you don't want to specify a data structure that loops back on itself (e.g. a linked list).
